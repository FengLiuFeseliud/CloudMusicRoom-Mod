# CloudMusicRoom-Mod [Fabric]

#### 服务端与 [CloudMusic](https://github.com/FengLiuFeseliud/CloudMusic-Mod) 交互实现一起听与点歌

#### 客户端 前置: MaliLib, CloudMusic, Fabric API

#### 服务端只需放入本 mod 进 mod 目录！！！
